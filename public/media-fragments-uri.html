Media Fragments URI

A temporal dimension for the video is specified.

Open the Network tab of your browser dev tool and see the HTTP headers `Range` in the request and `Content-Range` in the response.

Also, please note that each video segment has a status code of `206 Partial Content`.

Depending on the video file and the browser, there is either:

  • A single request loading the complete video.
  • A first request, aborted, to get the header, then one or more requests to get the section needed.
